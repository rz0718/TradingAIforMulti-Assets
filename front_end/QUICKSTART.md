## Running the Dashboard

### Option 1: Local Access
```bash
./run_dashboard.sh
```

### Option 2: Network Access (Share on WiFi)
```bash
./run_dashboard_network.sh
```

### Option 3: Internet Access (Share with anyone)
```bash
./run_dashboard_ngrok.sh
```
